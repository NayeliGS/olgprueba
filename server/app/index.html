¿Qué es la salud visual?
La salud visual abarca tanto el bienestar, como el óptimo funcionamiento del sistema ocular.
Una buena salud visual es fundamental para la calidad de vida, permitiéndonos tener una interacción eficiente con nuestro entorno y previniendo problemas que puedan limitar nuestra capacidad de ver y percibir el mundo que nos rodea.
